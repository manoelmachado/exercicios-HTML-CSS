Listas não ordenadas

Essas são as litas não ordenadas, onde conseguimos destacar itens sem uma ordem específica. Usamos a tag <ul> (unordered list). Exemplo:

  • Item 1
  • Item 2
  • Item 3

Podemos também alterar o ícone da lista para um circulo, um quadrado ou uma bolinha(padrão) usando type="square/circle/disc dentro da tag ul

Listas ordenadas

Funciona da mesma maneira que a lista não ordenada, mas essa lista serve para ter uma sequência que precisa ser feita, como uma receita ou um passo-a-passo de algo.

  1. Passo 1
  2. Passo 2
  3. Passo 3

Também podemos alterar o ícone da lista para números romanos, letras ou números(padrão) usando type="1/A/a/I/i

Listas dentro de listas:

Da mesma maneira que criamos uma lista, podemos criar outra lista dentro da lista criada.

  1. Lista 1
    • Lista 1.1
    • Lista 1.2
    • Lista 1.3
  2. Lista 2
    • Lista 2.1
    • Lista 2.2
  3. Lista 3
    • Lista 3.1
    • Lista 3.2

Lista de definições:

Listas de definições são geralmente usadas para ter uma descrição do termo usado. Usamos a tag <dt>(Termo) e <dd>(Descrição)

Termo
Descrição
HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para criação do design de um site.
JavaScript
Linguagem de programação para criação da interatividade de um site.
