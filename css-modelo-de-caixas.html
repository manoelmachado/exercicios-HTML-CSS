Exemplo de caixa box-level

Parágrafos também são exemplos de box-level, mas os links são exemplos de caixas inline-level. Vamos ver como tudo isso funciona.

Todo elemento do HTML é renderizado pelo navegador como uma box. Essa box pode ser alterada com as propriedades CSS: margin, border, padding e height/width. O margin nada mais vai movimentar o elemento, para a esquerda, direita, para cima ou para baixo. O border é a borda, dá pra definir o tamanho de borda, cor e modo de exibição. O padding é o preencimento, ele vai colocar o um espaçamento dentro do elemento. Width/height representam o conteúdo do elemento, um texto, uma imagem, um vídeo....

Shorthand:

Podemos usar shorthands para simplificar o código CSS, lembrando que é necessário estar em ordem, caso contrário não funcionará.

  • shorthand para border: width style color
  • shorthand para padding: top right bottom left
  • shorthand para margin: top right bottom left
  • shorthandpara outline: width style color

Observações importantes:

Podemos alterar o modelo da caixa de block ou para inline usando display: inline ou block para transformar em uma caixa inline ou blokc respectivamente.

Podemos deixar a caixa centralizada na tale utilizando o "margin: auto;"
